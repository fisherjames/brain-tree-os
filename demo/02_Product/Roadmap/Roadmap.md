# Roadmap

Development timeline for clsh.dev.

## Completed

### Week 1: Foundation to Launch (Days 1-6)
- Day 1-2: Project setup, core terminal, basic UI
- Day 3-4: Mobile keyboard system, auth, tunneling
- Day 5: Security audit, marketing prep, community setup
- Day 6: Launch on Reddit, Product Hunt, Hacker News

### Post-Launch (Week 2)
- Bug fixes from user feedback
- Performance improvements
- Documentation updates

## In Progress

### Q2 2026: Stabilization
- Mobile UX iteration based on real usage data
- Performance optimization for slow connections
- Accessibility improvements

## Planned

### Q3 2026: Multi-User (v2)
- Team access with role-based permissions
- Shared terminal sessions
- Session recording and playback

### Q4 2026: Cloud Hosted (v3)
- Managed cloud machines
- Persistent environments
- CI/CD integration

All roadmap decisions align with [[MVP]] scope.
