# Go-to-Market Strategy

Launch and growth plan for clsh.dev.

## Launch Channels (Priority Order)

1. **Reddit** (r/commandline, r/programming): Developer communities
2. **Hacker News** ("Show HN"): Technical credibility
3. **Twitter/X**: Developer network effects
4. **Product Hunt**: Broader tech audience

## Messaging

### Primary
"Access your Mac terminal from your phone. One command, no setup: `npx clsh`"

### Technical
"xterm.js + node-pty + ngrok. Real PTY, custom mobile keyboard, zero config."

## Funnel

```
Awareness (Reddit/HN)
  → Interest (clsh.dev)
    → Trial (npx clsh)
      → Adoption (daily use)
        → Advocacy (star, share)
```

## Growth Levers

- Open source virality via GitHub
- Community word-of-mouth
- [[SEO]] for organic discovery
- [[Social-Media]] for ongoing engagement
