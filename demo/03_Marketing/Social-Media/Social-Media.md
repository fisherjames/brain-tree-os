# Social Media

Platform strategy and presence for clsh.dev.

## Active Platforms

### Discord
Primary community hub with 5 categories, 14 channels, 3 roles. Active support and feature discussions.

### Twitter/X (@clsh_dev)
Developer audience engagement with technical content and launch updates.

### Reddit
High-impact launch platform across r/commandline, r/programming, r/selfhosted.

### GitHub
Open source presence with README as marketing surface and Discussions for feedback.

| Platform | Frequency | Content Type |

| Twitter/X | 3x/week | Updates, tips |
| Reddit | Monthly | Major releases |
| GitHub | Continuous | Releases, docs |
