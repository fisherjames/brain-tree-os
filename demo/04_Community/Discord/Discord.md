# Discord Server

Primary community hub for clsh.dev users and contributors.

## Server Structure

### Categories & Channels

**Welcome** (2 channels)
- #welcome: Rules and getting started
- #introductions: New member intros

**General** (3 channels)
- #general: Main discussion
- #showcase: User terminal setups
- #off-topic: Casual chat

**Support** (3 channels)
- #help: Troubleshooting
- #bug-reports: Issue reporting
- #feature-requests: Ideas and votes

**Development** (3 channels)
- #dev-chat: Contributor discussion
- #pull-requests: PR notifications
- #releases: Version announcements

### Roles
- **Maintainer**: Core team, full access
- **Contributor**: Active contributors
- **Community**: All members, default

## Moderation

- AutoMod for spam and link filtering
- Welcome bot with onboarding message
- GitHub integration for PR/issue notifications
