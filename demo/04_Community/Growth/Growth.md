# Growth Strategy

Community growth and user acquisition for clsh.dev.

## Growth Channels

### Organic (Primary)
1. GitHub trending page through consistent stars
2. Reddit engagement in developer communities
3. Word of mouth from delightful product experience
4. Developer blog guest posts

### Community-Driven
1. Discord weekly "terminal tips" sessions
2. Contributor pipeline via good first issues
3. Showcase channel for user setups
4. Integration partnerships with terminal tools

## Growth Metrics

| Metric | Week 1 | Month 1 | Month 3 Target |
|--------|--------|---------|----------------|
| GitHub Stars | 500 | 1,000 | 3,000 |
| Discord | 100 | 300 | 1,000 |
| npm/week | 200 | 500 | 2,000 |

## Flywheel

More users → More feedback → Better product → More word-of-mouth → More users
