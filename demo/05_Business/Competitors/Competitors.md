# Competitive Landscape

Analysis of alternatives in the remote terminal space.

## Direct Competitors

| Product | Type | Mobile | Price | Setup |
|---------|------|--------|-------|-------|
| **clsh.dev** | Browser terminal | Native | Free | npx clsh |
| Termius | SSH app | App Store | Freemium | Manual |
| Blink Shell | SSH (iOS) | iOS native | $20 | Manual |
| Prompt 3 | SSH (macOS/iOS) | iOS native | $40 | Manual |

## Differentiation

### What We Do Differently
1. Zero config: No SSH key management, no server setup
2. Custom keyboards: Built for terminal, not adapted
3. Browser-based: Works on any device, no app install
4. Open source: Transparent, extensible
5. Free: No subscription for core features

### Where They Win
1. Persistent connections and multi-server support
2. Native app performance
3. Established trust and reputation

We're solving a different problem: "terminal right now, from phone, zero setup."
